## 🌐 Overview

**Onframe** is a next-generation platform designed to empower builders, creators, and developers to launch products faster using AI-driven workflows.

Instead of starting from scratch every time, Onframe enables continuous growth — every product you ship improves your future builds.

## 🚀 Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/your-username/onframe.git
cd onframe
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run Development Server

```bash
npm run dev
```
